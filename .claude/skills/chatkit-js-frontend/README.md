# ChatKit JS Frontend Skill

This skill provides comprehensive guidance for building frontend chat interfaces using OpenAI's ChatKit library.

## Overview

The chatkit-js-frontend skill helps you:
- Set up ChatKit in React applications
- Configure authentication and session management
- Customize the chat UI and user experience
- Handle events, errors, and analytics
- Integrate with backend services like FastAPI

## Components

- `SKILL.md` - Main skill documentation with usage instructions
- `references/api-reference.md` - Detailed API documentation
- `references/fastapi-backend.md` - FastAPI backend integration examples
- `scripts/setup-chatkit-react.sh` - Utility script to set up a new ChatKit project

## Usage

When you need to implement a ChatKit frontend:

1. Use the code examples from `SKILL.md` as starting points
2. Refer to `api-reference.md` for detailed API documentation
3. Check `fastapi-backend.md` for backend integration patterns
4. Use `setup-chatkit-react.sh` to bootstrap new projects

## Key Features

- React integration with `useChatKit` hook
- Session management and authentication
- Theme customization
- Event handling and analytics
- Widget action support
- Error handling and recovery
- FastAPI backend integration examples
